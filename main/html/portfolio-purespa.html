User Stories

As a user:

  • I want detailed descriptions of the treatments and facilities offered so I can make an informed choice.
  • I want a quick and easy booking process for treatments, so I can schedule with ease.
  • I want transparent pricing for services, so I can budget appropriately for my visit.
  • I want to know the opening and closing hours, so I can plan my visit accordingly.
  • I need clear location and contact information for the spa center, so I can make questions or book an appointment.

Color Palette

The colour scheme was chosen with the main color a shade of green, which stands for balance, nature and rebirth, exacly what our website represent and intended to envoke prosperity and freshness to the potential spa costumers.

Main colours used:

  • background color: #F6F6F6 and #33472B
  • navigation-bar and footer: #1F2324
  • hover color: #33472B

Fonts

  • Main content font-family: Roboto, sans-serif
  • Headings and subheadings: EB Garamond, serif;

Ideation

Based on user stories and research from other spa centers, low-fidelity wireframes were created to capture a high-level concept for the website.

Performance Testing

Our website has passed comprehensive performance testing ensuring:

  • Fast load times and optimal responsiveness
  • Accessibility standards, providing an inclusive experience for all users
  • Code efficiency to ensure maintainability and scalability
  • SEO fully optimised, with proper metadata, fast load times, and mobile-friendly design to improve visibility and search engine rankings.
